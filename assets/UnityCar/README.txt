UnityCar 2.1 Pro

Package content:

Assets/UnityCar/UnityCarScript
this folder contains all the UnityCar scripts.
In order to use UnityCar in your projects you need to copy this folder in the script folder of your project.
In order to create a basic full functional car just select "UnityCar" under the menu GameObject/Create Other/UnityCar.


Assets/UnityCar/Editor
this folder contains all the UnityCar editor scripts.
These scripts are used to show the wiki and setup load/save buttons in UnityCar scripts inspector. Copy this folder in the editor folder of your project to show wiki and setup load/save buttons in the ispector.



Assets/UnityCar/Samples
this folder contains all the Sample Assets you may need to test UnityCar.
Included Sample Assets:

Cars (in Assets/UnityCar/Samples/Prefabs/Cars):
1) F360 (powerful RWD sport car, made by sipon for UnityCar)
2) MP4 (powerful RWD sport car)
3) R34 (sport RWD car, made by deth_moroz for UnityCar)
4) Kooper (sport FWD city car)
5) RGK Truck (offroad AWD car)
6) Monster Truck (big wheel AWD monster truck, made by sipon for UnityCar)
7) SC Truck with Trailer (RWD truck)
8) TukTuk (3-wheeled RWD car)
9) Moon Buggy (6-wheeled AWD car)

All models are royalities free.


Tracks (in Assets/UnityCar/Samples/Prefabs/Tracks):
1) Monza Track (made by sipon for UnityCar)
2) Sipon City (made by sipon for UnityCar)
3) Pixelactive City (provided by ikiman under his conditions. Not for commercial purposes)
4) Stress Test (a simplified version of Pixelactive City)
5) Rally Sweden (made by sipon for UnityCar)
6) Tutorial Track (taken from Unity Car Tutorial)



Assets/UnityCar/Scenes
this folder contains all the UnityCar demo scenes:
1) Monza Track
2) Sipon City
1) Pixelactive City 
2) Stress Test
3) Rally Sweden 
4) Tutorial Track 



Assets/UnityCar/InputManagerSettings
this folder contains the file "InputManager.asset". You have to copy it in the of your project ProjectSettings folder (if you have a PC with Unity 3.4 backward in Library folder) in order to import UnityCar Input Settings (steer, throttle, brake, gear shift).



Assets/UnityCar/UnityCarLogo
this folder contains the official UnityCar Pro logo in psd format.



Assets/UnityCar/Plugins
this folder contains the file UnityForceFeedback.dll (made by Vectrex, fixed by rogeliopuig). You have to put it in Assets/Plugins folder of your project in order to enable Force Feedback on devices like Logiech G25-G27.
